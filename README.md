# Hallowest

Hallowest is a gameplay overhaul mod for Hollow Knight that increases the overall difficulty of Hallownest. The mod focuses on making combat more challenging and engaging while preserving the original game feel.

## Overview

This mod modifies core gameplay systems to create a more demanding experience. Enemies become stronger, bosses are more dangerous, and combat encounters require more precision and awareness.

The goal of Hallowest is not to change Hollow Knight into a different game, but to push its existing systems to a harder and more punishing version of themselves.

## Features

- Increased enemy difficulty
- Buffed boss encounters
- Faster and more aggressive AI behavior
- Adjusted attack patterns for higher challenge
- Global combat difficulty scaling

## Planned Features

- Platforming difficulty enhancements
- Additional environmental hazards
- Further balance refinements across Hallownest

## Installation

1. Install Lumafly
2. Download the latest release of Hallowest
3. Enable the mod in the Lumafly

## Compatibility

Hallowest is designed to work with a clean gameplay experience. Compatibility with other gameplay overhaul mods is not guaranteed.
